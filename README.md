# Bittensor Workshop

Repositori pembelajaran dan eksplorasi ekosistem **Bittensor** — jaringan terdesentralisasi untuk machine intelligence. Repo ini berisi materi belajar, referensi subnet template, dan catatan persiapan hackathon.

## Isi Repo

| Folder / File | Deskripsi |
|---|---|
| `bittensor-subnet-template/` | Template resmi untuk membangun subnet Bittensor (miner, validator, protocol) |
| `hackathon bittensor.md` | Catatan lengkap tentang Bittensor Subnet Ideathon (timeline, submission, hadiah) |

## Bittensor Subnet Template

Template dasar untuk membuat subnet di jaringan Bittensor, mencakup:
- **Miner** — node yang mengerjakan tugas komputasi
- **Validator** — node yang menilai hasil kerja miner
- **Protocol** — definisi komunikasi antara miner dan validator
- **Reward mechanism** — logika pembagian insentif

Dokumentasi lebih lanjut: [`bittensor-subnet-template/README.md`](bittensor-subnet-template/README.md)

## Hackathon: Bittensor Subnet Ideathon

Hackathon global oleh **Bittensor x HackQuest** untuk mendesain dan membangun subnet Bittensor.

- **Platform**: [HackQuest](https://www.hackquest.io/hackathons/Bittensor-Subnet-Ideathon)
- **Round I** (Ideation): 22 Des 2025 → 25 Feb 2026
- **Round II** (Hackathon): 1 Mar → 30 Mar 2026
- **Hadiah**: Pitch session dengan Co-Founder Bittensor + potensi investasi hingga 1,000 TAO

Detail lengkap: [`hackathon bittensor.md`](hackathon%20bittensor.md)
